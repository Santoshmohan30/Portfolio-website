How I Update This

Working flow for adding a new post

Create a file in `posts/`

Copy `weekly-template.html`, rename it, and write the full post there.

Update both listing pages

Add one blog card here in `blog/index.html` and one in the homepage preview inside `index.html`.

Files To Touch

`posts/your-new-post.html`, `index.html`, and `blog/index.html`.

That is the whole workflow.

Use the template system in `/posts` to add new writing quickly.
