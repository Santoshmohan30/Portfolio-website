How I Update This

Working flow for adding a new post

Create a file in `posts/`

Copy `weekly-template.html`, rename it, and write the full post there.

Update one metadata file

Add one object to `assets/js/blog-data.js` and both the homepage and blog archive update automatically.

Files To Touch

`posts/your-new-post.html` and `assets/js/blog-data.js`.

That is the whole workflow. Nobody can publish anything unless they have access to your GitHub repo.

Use the work update template in `/posts` to add new writing quickly.
